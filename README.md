# Alltools

Alltools 是一个基于 **原生 HTML / CSS / JavaScript** 构建的个人工具导航网站，用于集中管理和分类展示个人常用工具与在线资源。项目采用纯静态实现，不依赖任何前端框架，结构清晰，易于维护和扩展，适合长期作为个人技术门户或工具入口。

## 项目背景

在日常学习、开发与研究过程中，会频繁使用各类在线工具和自研工具。工具分散在不同网站中，不利于统一管理和快速访问。Alltools 的目标是将这些工具进行集中整理，通过清晰的分类与简洁的界面，实现高效、稳定、可持续维护的工具导航站点。

## 功能特性

- 工具按功能分类展示
- 每个工具包含名称、简要说明及访问链接
- 支持一键跳转至外部工具页面
- 数据与页面结构解耦，便于维护
- 纯静态页面，可直接部署到 GitHub Pages 或服务器

## 技术栈

- HTML5：页面结构与语义化布局
- CSS3：页面样式与响应式布局
- JavaScript（Vanilla JS）：数据管理与动态渲染

不依赖任何第三方框架或库，确保项目轻量、可控。
